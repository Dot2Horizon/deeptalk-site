How to Export Chat Files

WhatsApp

1

Open the chat you want to export

2

Tap the chat name at the top

3

Scroll down and tap Export Chat

4

Select Without Media

5

Share the .txt file to DeepTalk

KakaoTalk

1

Open the chat room

2

Tap the menu (hamburger icon) at the top right

3

Tap the settings gear icon

4

Select Export Chat

5

Share the exported file to DeepTalk

LINE

1

Open the chat room

2

Tap the menu icon at the top right

3

Tap Other settings

4

Select Export chat history

5

Share the .txt file to DeepTalk

FAQ

Is my chat data safe?

Yes. All analysis happens 100% on your device. Your chat data is never uploaded to any server. We cannot see your messages even if we wanted to.

What happens to my data when I delete the app?

All imported chat data and analysis results are permanently deleted when you uninstall DeepTalk. Nothing remains on your device or on any server.

Why does the app need network access?

Network access is used only for anonymous usage analytics (TelemetryDeck) and crash reporting (Firebase Crashlytics). No chat data or personal information is ever transmitted.

How many messages can DeepTalk handle?

DeepTalk is optimized for large chats and can handle 100,000+ messages efficiently using streaming parsers and batch processing.

Which file formats are supported?

DeepTalk supports .txt and .zip files exported from WhatsApp, KakaoTalk, and LINE. The messenger format is automatically detected.

Does the AI Chat Insight require internet?

No. The AI runs entirely on your device using a local language model. No internet connection is needed for any analysis feature.

Can I analyze chats from multiple messengers?

Yes. You can import chats from WhatsApp, KakaoTalk, and LINE independently. Each import creates a separate analysis.

The app is slow during analysis. Is this normal?

Initial analysis of large chat files may take a moment since all processing runs on-device. Once complete, all analytics cards load instantly.
